# Exercise Website

This repository contains a minimal black and white exercise website. Open `site/index.html` in a browser to use it.

## Features
- Home page with buttons for Walk, Treadmill, Stairs, Upper Body, Lower Body and Core.
- Each category page lists saved routines and lets you add new ones.
- Treadmill, Stairs and Walk use an interval template (overall duration, interval duration and pace).
- Upper Body, Lower Body and Core use a reps template (exercise name, reps and link to a demo).
- Routines are stored locally in your browser.
